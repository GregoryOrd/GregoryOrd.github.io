Lab 1 - Fractal Geometry

This was the first application built as part of the "CPSC 453 - Introduction to Computer Graphics" Course. We needed to create four scenes and generate the geometry for a new fractal shape in each scene. Keyboard inputs were used to switch between scenes as well as change the number of iterations drawn for each scene.


The four fractal shapes we needed to generate for this assignment were:

  1. Serpinsky Triangle
  2. A Diamonds and Squares Pattern
  3. The Koch Snowflake
  4. The Dragon Curve
