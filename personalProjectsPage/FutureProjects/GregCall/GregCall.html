GregCall - A VoIP Solution for Audio Calls

Project Description

I was quited excited when I worked on the RTP Proxy project to support a Voice over IP solution at Calian. I thought the technologies used for voice calls were super cool and wanted to try making my own solution at home.

I made a git repo and setup a hello world program as an initial commit for the project, but ended up getting busy and never starting on the project.

My idea for this project is to use Session Initiaion Protocol (SIP) to setup a connection between to clients and then start streaming Real Time Transport Protocl (RTP) packets between the clients after a session has been started.
