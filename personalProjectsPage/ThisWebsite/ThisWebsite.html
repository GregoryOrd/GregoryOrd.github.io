Project Description

I started working on this website in June 2019 while I was working at the Calgary Counselling Centre. I was hoping that it could be an opportunity to learn about HTML/CSS/Javascript and as be a good place to show off my projects and experience when I started applying for internships for the next year. All of the HTML/CSS/Javascript in this website is writen by me, except for the Bootstrap 5 CSS classes and Font Awesome icons.

Over time I have had more work experiences and completed more personal projects, so the site has needed to be updated. Some of the changes in layout over time are shown below. Originally the personal projects and work experience sections were together. Then as more projects and experience were added, they were split apart to not have such a large page. Then as more time went on, even my personal projects were added, so that page was split up into groupings by ranges of years.

As all of this has happened, the code has been cleaned up and refactored to make it easier to scale and maintain.

Also, some of the interesting features of this website are the image carousels and the image viewer. Both of these features were built into custom HTML components that can be reused across this site simply by using tags in the HTML (<image-carousel></image-carousel> and <viewable-img></viewable-img>). Most of the images throughout the project use this viewable image element and can be clicked on to open a view modal that allows the user to zoom and pan around the image.
