Dinos Inspire Training Log Website (June to August 2017)

Project Description

This was a weekend warrior type project that I worked on during the summer of 2017. This project was a web-based application, built using PHP, that allowed distance runners to create accounts and then store data related to their training in a database. The inspiration for this project came from my involvment in the sport and awareness of the importance of keeping a proper training log. I viewed the project as an opportunity to learn about the technology and get a sense for how other training log websites, like Garmin Connect or Strava, might work.

This project did not reach completion by the end of summer 2017 and got put on the backburner when school begin in September. However, I did feel that I took this project far enough to achieve the goal of learning the basics of how client-side and server side languages work together in a web application to navigate from page to page, pass data, and interact with a database.



Lessons Learned

Looking back at this project it is interesting to see how many things I could have done better, and how much I have learned since then. From a user experience point of view, the two main area for improvement are mobile responsiveness and dynamic display of formatted run/workout data. These are areas that I feel I have greatly improved upon with the my work at the Calgary Counselling Centre and with this personal portfolio website. If I were to do the training log project again a lot of things would be different, but overall I think it was a good learning experience.
