Air Monitor MCP Server - An MCP Server Allowing Providing Data From an Air Monitor to Local LLMs (Start Date to End Date)

Project Description

An MCP Server used to provide data from an AirGradient indoor air quality monitor to local LLMs.

This project is currently in early stages with only two disconnected components. The first component is a simple java program that regularly polls the sensor and writes the data to an influx database.

The second component is a Gemma 4 LLM running with Ollama inside a docker container.

The MCP server to connect the two is the exciting part that is still to come.
