This program read from the log file of the named daemon, parsed out the relevant DNS log data, and wrote it to our postgres database. The postgres table was then displayed in a GUI using code that Calian had already developed.

For this program I used the logrotate tool and install pre/post rotate scripts to detect rotations in the named.log file. The program was written in python and used linux signal handlers to receive signals of the logrotatation events from the pre/post rotate scripts. The state of the DNS log reader was also maintained in sqlite DB files so that the program could behave properly on start and restart and could handle down time without any loss up until a reasonable duration of time.

To control execution of the program we setup a service file and ran it with systemd.
