My Crush on C/C++


Born and raised in Ontario, I moved to Calgary to study at the University of Calgary and run for the Dinos Cross-Country team in 2016.

As an athlete, I was fortunate to win the 2016 Canada West Cross-Country Rookie of the Year award and be surrounded by great teammates. Together our hard work lead to the team winning silver at the 2017 USPORTS National Championships, and gold at the 2018 and 2019 editions.

My first programming experience was in a first year course, ENGG 233 - Computing for Engineers. A couple years later, I gained my first software development work experience at the Calgary Counselling Centre in the summer of 2019. Then a 16-month internship at Lockheed Martin CDL Systems from May 2020 to August 2021, and since May 2022, a fulltime position at Calian Advanced Technologies.

I have used different languages over the years, and worked in both Windows and Linux development environments, though I do much prefer working with Linux. While working with control station software for unmanned vehicles during my Lockheed Martin internship, I became very interested in C++, the build system, and the unit testing framework they were using to build such a large project. This led to a couple of personal side projects which I called GregBuild and GregTest.

A little while later, the idea came up to try my hand with a little bit of embedded device development. I had been using Garmin running watches for years, so I decided to try building one. At the time I had a Raspbeery Pi zero laying around and my girlfriend had an Arduino Uno that she wasn't using, so GregBuild was addapted to use a plugin architecture with dynamically linking libraries and a plugin was build for cross-compiling on the Raspberry Pi and flashing the firmware onto the Arduino Uno. I attached a screen to the Arduino, and came up with my own pixel schemes for the number's of a digital clock face. GregBuild then became the build system for the C code used to create the watch project which I called GregWatch.

The GregWatch code got to a point I was happy with, and I designed a PCB to replace the Arduino. When the PCB arrived from Osh Park, there were some problems with it, and then life got busy and the work on GregWatch ended there. But it was a fun project to work on, and I thought that it was really cool that I was able to prove the usefulness of GregBuild by building the code for GregWatch with it.

Since then, I have been working at Calian Advanced Technologies. With Calian I have learned a bunch more about how Linux servers work and writen some documentation for our customers' system administators. For Calian, I've worked in C/C++, Java, python, writen upgrade and task automation scripts in bash, and even created an ISO image with all of the packages and system configuration we needed when updating an operating system. When I look back, I realize I've learned a lot since that first course in 2016, and I am eager to continue learning as my career progresses!
